### 🔄 **Azure Active Directory (Azure AD)** has been **rebranded to**:

> **🔵 Microsoft Entra ID**

---

### 📢 **Why the change?**

Microsoft renamed **Azure AD to Microsoft Entra ID** to align it with the broader **Microsoft Entra** family of identity and access products. It’s a **name change only** — the features, capabilities, and pricing remain the same.

---

### 📦 **Microsoft Entra Product Family**:

|Product|Description|
|---|---|
|**Microsoft Entra ID**|Formerly Azure AD — identity and access management|
|**Microsoft Entra Permissions Management**|Cloud infrastructure permissions governance|
|**Microsoft Entra Verified ID**|Decentralized identity (blockchain-like credentials)|
|**Microsoft Entra ID Governance**|Automate access lifecycles, access reviews, etc.|

---

### ⚙️ What’s **not changing**:

- Your **existing configurations**
- **APIs**, **PowerShell** commands (e.g., `az ad` still works)
- Azure portal menus (they’ll reflect the name change gradually)
- Licensing (Free, P1, P2 tiers remain same)

---

### ✅ Example Transition:

|Old Term|New Term|
|---|---|
|Azure Active Directory|Microsoft Entra ID|
|Azure AD Premium P1|Microsoft Entra ID P1|
|Azure AD B2C|Microsoft Entra External ID (coming soon)|

---

### 🧑‍💻 Developers & Admins:

You can still use:

```bash
az ad user list
az ad group create
```

These commands will continue to work even after the name change.
